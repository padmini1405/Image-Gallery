# Image Gallery (React)

A simple and responsive Image Gallery built using ReactJS.  
This project demonstrates component-based architecture, dynamic rendering, and responsive design.

## Features

- Dynamic image rendering using `map()`
- Reusable components (ImageCard)
- Data stored in an array of objects
- Props used for passing data
- Fully responsive (Desktop, Tablet, Mobile)
- Clean UI with modern CSS styling
- Fast and lightweight

## Tech Stack

- React JS
- JavaScript (ES6)
- HTML5
- CSS3
